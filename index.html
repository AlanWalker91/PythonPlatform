学习方式
  • 先讲背景、来源和痛点

知识体系
  • 基础：复杂度、数组、链表、栈、队列
  • 查找：哈希表、二分查找、双指针
  • 进阶：树、堆、图、滑动窗口、回溯、动态规划
  • 专题：贪心、前缀和/差分、单调结构、并查集、Trie
  • 目标：会讲、会写、会分析、会应用
学习顺序
  • 第 1 阶段：复杂度 -> 数组 -> 链表 -> 栈队列
  • 第 2 阶段：哈希 -> 二分 -> 双指针 -> 滑动窗口
  • 第 3 阶段：树 -> 堆 -> 图搜索
  • 第 4 阶段：回溯 -> 动态规划
  • 第 5 阶段：贪心 -> 前缀和/差分 -> 单调结构 -> 并查集/Trie
