# Python Platform

一个面向 Python 数据结构与算法学习的可视化教学平台，目标不是只给题解，而是把“背景、痛点、核心思想、代码实现、复杂度分析、练习反馈”串成一条完整的学习主线。

这个项目更偏向“课程产品 + 算法练习平台”的结合体，适合：

- 准备 Python 算法面试的同学
- 想系统补齐数据结构与算法体系的初学者
- 希望把“会看题解”升级到“会讲、会写、会分析”的学习者

---

## 项目亮点

### 1. 结构化课程设计

每个章节都按统一教学节奏组织：

- 背景与来源
- 典型痛点
- 核心思想
- 经典例题
- 代码拆解
- 算法思路总结
- 模板代码
- 实战练习

这样做的目的，是让学习不再停留在“看懂这道题”，而是逐步形成自己的题型识别能力。

### 2. 面试导向的知识体系

当前课程已经覆盖：

- 复杂度分析
- 数组与 `list`
- 链表
- 栈与队列
- 哈希表
- 二分查找
- 双指针
- 滑动窗口
- 树与二叉树
- 堆与优先队列
- 图、BFS 与 DFS
- 回溯
- 动态规划
- 贪心
- 前缀和与差分
- 单调栈与单调队列
- 并查集与 Trie

整体学习顺序遵循“先基础、再套路、再搜索、再优化、最后专题深化”的递进路径。

### 3. 可视化算法演示

平台内置动态演示模块，帮助学习者理解那些“只看代码很难一下子想明白”的过程：

- 复杂度增长趋势
- 线性扫描
- 二分区间收缩
- 树遍历
- 堆上浮
- 图搜索
- 双指针收缩
- 滑动窗口扩张与修复
- 回溯搜索树

其中一部分章节已经升级为 SVG 图形化演示，支持节点、连线、路径高亮和区间指针。

### 4. 实战练习而不是选择题

每章结尾都提供 LeetCode 风格的 Python 练习：

- 同章节多题切换
- 直接写函数代码
- 运行公开样例与隐藏测试
- 看通过 / 未通过反馈
- 查看提交详情与代码快照
- 配套章节提示与解题方向

目标是把“理解”真正转成“能写出正确代码”。

### 5. 可持续学习能力

除了课程内容本身，平台还支持本地学习状态管理：

- 章节完成状态
- 错题本
- 错题二刷模式
- 最近提交记录
- 提交记录页
- 学习导航建议
- 本地进度保存
- 学习进度导出 / 导入
- 一键重置进度

这让它不只是一个演示页，而是可以长期使用的学习工作台。

## 本地运行

### 方式一：直接双击启动脚本

项目目录下提供了启动脚本：

`启动PythonPath平台.bat`

它会启动本地 Python 服务，随后打开平台页面。

### 方式二：手动启动

1. 进入项目目录
2. 启动本地服务
3. 在浏览器中打开对应地址

示例：

```bash
python server.py
```

如果你只想看静态页面，也可以直接打开 `index.html`，但实战练习的运行测试需要本地服务支持。

## 适合怎么学

如果你是新手，建议按照下面的顺序推进：

### 第 1 阶段：打基础

- 复杂度分析
- 数组与 `list`
- 链表
- 栈与队列

目标：先建立“数据怎么组织、代码为什么快慢不同”的底层感觉。

### 第 2 阶段：掌握高频套路

- 哈希表
- 二分查找
- 双指针
- 滑动窗口

目标：建立最常见的面试解题框架。

### 第 3 阶段：进入搜索与结构

- 树与二叉树
- 堆与优先队列
- 图、BFS 与 DFS

目标：理解递归、层次搜索、优先级维护等更复杂结构。

### 第 4 阶段：训练抽象能力

- 回溯
- 动态规划

目标：从“写代码”逐步走向“设计算法”。

### 第 5 阶段：专题突破

- 贪心
- 前缀和与差分
- 单调结构
- 并查集与 Trie

目标：补齐高频专项，提升题型识别精度。
